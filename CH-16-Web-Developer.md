## Article

1. **Role Description** – An Alchemy Alliance {Web Developer} designs the website where Alchemy Alliance content is hosted.

2. **Responsibilities** – Alchemy Alliance {Web Developer}s design scripts and functions that accept images, spreadsheets, and text files, loading into the Alchemy Alliance website. NOTE: This is not a role that entails populating the website with content, but rather designing the optimal way for that content to be uploaded and displayed. Provides occassional support to others who add content to the website.

3. **Time Commitment** – This is passive role, awarded because an individual has deep familiarity with how the website is set up and could help troubleshoot errors while slowly adding more features.

4. **Benefits** – The Alchemy Alliance is run by volunteers. There is no monetary compensation currently available for this role. Instead it is an opportunity to showcase technical skill with websites, while supporting a "good cause".

5. **Qualifications** – Applicants must:
   * Be an Alchemy Affiliate per CH-06.
   * Have a free [GitHub Account](https://github.com).
   * Have experience in website development.

6. **Desired Qualities** – Advantage to also:
        * Have experience with populating content on the website.

7. **On/Off Boarding**
Per CH‑06‑Affiliate.
