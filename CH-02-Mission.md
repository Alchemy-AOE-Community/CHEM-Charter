## Article

1. **About** – The Alchemy Alliance is a community dedicated to innovating and sharing custom random map experiences for Age of Empires II, fostering creativity while respecting intellectual property boundaries.

2. **Intellectual Property** – Age of Empires II (AOE2) is a real-time strategy game ("the game") owned by the Microsoft Corporation. The coding required to achieve the following is trade secret and intellectual property of Microsoft:
    * Distribution
    * Design (units, civilizations, technologies, objects, artwork, map features, etc.)
    * Multiplayer
    * Any other features or assets native to the game as presented by the developer.

3. **The Materials** – The Alchemy Alliance creates high-level random map customizations ("The Materials") using external editors designed to function within the Age of Empires II (AOE2) environment. 
    <ol type="A">
      <li><strong>Ownership:</strong> The Materials are derivative customizations owned by their creators, in the same way that a book written electronically is owned by its author, and not the software developer of the word-processor.</li>
      <li><strong>Distribution:</strong> The Materials are distributed outside of official Age of Empires II forums.</li>
    </ol>

4. **Donations** – Publicly tracked donations are used solely to support non-commercial community events unaffiliated with Microsoft.

5. **Recognized File Types** – File types recognized by the Alchemy Alliance are summarized below:

    | Description | Extensions | Developmental Relationship With AOE2 | Intellectual Property Ownership | Part of the Materials? |
    | :--- | :---: | :--- | :---: | :---: |
    | Recorded Game File | .aoe2record | Generated whenever a game of AOE2 is played | Microsoft | No |
    | Scenario File | .aoe2scenario | Created through AOE2's scenario editor | Microsoft | No |
    | Random Map Scripts | .rms | Written in external editors, inspected using the game | Map Author | Yes |
    | Script Compilation Tools | .m, .ods, etc. | Written in external editors, inspected using the game | Map Author/Alchemy Alliance | Yes |
    | Specifications | .odt, .pdf, etc. | Written in external editors, no game access required | Alchemy Alliance | Yes |
    <p align="center"><em><u>Table 1: Recognized File Types and Relationship with the Alchemy Alliance</u></em></p>

6. **Specification Types** – Specifications prepared by the Alchemy Alliance are summarized below:

    | Type | Abbr. | Description | Alchemy-Specific? |
    | :--- | :---: | :--- | :---: |
    | Handbook | HDBK | High-level definition of Alchemy-organized event. | Yes |
    | Procurement Spec | PSRM | Defines a collection of random maps for competition. | Yes |
    | Component Spec | CSRM | Defines technical needs for each map in a pool. | Yes |
    | General Spec | GSRM | Defines common characteristics among component maps. | Yes |
    | Method | MTHD | Defines process for using Alchemy trade-secret tools. | Yes |
    | Standard | STND | Defines common characteristics for products/procedures. | No |

7. **Scope of Operations**
    <ol type="A">
      <li><strong>Endorsed Content:</strong> In commissioning or promoting The Materials, the Alchemy Alliance may create or endorse:
      <ol type="1">
      <li>Livestreams of 1v1s, team games, or community games.</li>
      <li>Tournaments & showmatches organization.</li>
      <li>Castings of above mentioned tournaments.</li>
      <li>Forum discourse regarding improvement of The Materials.</li>
      </ol>
      </li>
      <li><strong>Exclusions:</strong> The Alchemy Alliance does <strong>NOT</strong>:
      <ol type="1">
      <li>Enter competitions as a "clan" or "guild" (competitive exclusivity is not required).</li>
      <li>Create build order guides or strategic best-practices for AOE2.</li>
      </ol>
      </li>
    </ol>
